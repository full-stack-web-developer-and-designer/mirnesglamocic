# mirnesglamocic.com

Personal portfolio website of **Mirnes Glamočić** — a web developer focused on building modern, fast, and user-friendly websites and applications.

## 🚀 About the Project

This website serves as an online portfolio showcasing my work, skills, certificates, and experience in:
- Web development  
- WordPress development  
- PHP applications  
- Front-end technologies  
- Personal and client projects  

The goal of the website is to provide a clean and professional space where potential clients or employers can easily browse my projects and contact me.

## 🛠️ Tech Stack

The website is built using:

- **HTML5**
- **CSS3**
- **JavaScript**
- **jQuery**
- **PHP**
- **MySQL**
- **Hosting:** Globalhost

## 📂 Features

- Responsive design optimized for all devices  
- Showcase of personal and professional projects  
- Contact form / CTAs  
- SEO-friendly structure  
- Fast loading performance
